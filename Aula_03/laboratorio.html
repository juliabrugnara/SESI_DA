Tags Estudadas

Nesta página apresentamos algumas tags avançadas do HTML que ajudam a estruturar melhor o código.

1. Tag: <abbr>

Função: Define uma abreviação ou um acrônimo. Quando o usuário passa o mouse por cima, o texto completo aparece.

Exemplo de Código: A linguagem <abbr title="HyperText Markup Language">HTML</abbr> é essencial.

Resultado Esperado: A linguagem HTML é essencial. (Passe o mouse sobre a palavra HTML).

2. Tag: <kbd>

Função: Representa uma entrada de teclado. Serve para indicar ao usuário quais teclas ele deve apertar (muito útil em tutoriais de programação).

Exemplo de Código: Aperte <kbd>Ctrl</kbd> + <kbd>S</kbd> para salvar o seu código no VS Code.

Resultado Esperado: Aperte Ctrl + S para salvar o seu código no VS Code.

3. Tag: <meter>

Função: Representa uma medida escalar dentro de um intervalo conhecido, como o uso de disco ou relevância. Não deve ser usado para progresso de download.

Exemplo de Código: <meter value="0.8">80%</meter>

Resultado Esperado: Espaço ocupado no servidor da feira: 80%

4. Tag: <progress>

Função: Diferente do meter, esta tag representa o progresso de conclusão de uma tarefa, como o envio do portfólio no formulário.

Exemplo de Código: <progress value="45" max="100">45%</progress>

Resultado Esperado: Carregando currículo do aluno: 45%
