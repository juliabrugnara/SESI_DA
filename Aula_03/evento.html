InovaTech Brasil: A Programação

Como tudo começou

Nós percebemos que muitos alunos brilhantes não tinham espaço para mostrar o que sabiam criar. A InovaTech Brasil nasceu para inverter a lógica: em vez das empresas falarem o tempo todo, nós damos o microfone para os estudantes. As empresas vêm até nós para caçar os melhores talentos nas nossas competições e apresentações.

Tire suas Dúvidas (FAQ)

1. Qualquer aluno pode dar palestra?

Sim! Basta submeter o tema do seu projeto durante a inscrição. Os melhores serão selecionados para o palco principal.

2. Preciso ter experiência para as competições?

Não. As competições são divididas em níveis: Iniciante, Intermediário e Avançado. O foco é aprender!

3. Como entrego meu currículo para as empresas?

Você pode anexar seu currículo digital no formulário de inscrição, e as empresas terão acesso ao banco de talentos.

4. Tem custo para participar?

Para alunos de rede pública e universitários credenciados, o evento é 100% gratuito.

5. Posso ir apenas para assistir?

Com certeza! É uma ótima forma de se inspirar para os próximos anos.

6. Quais linguagens de programação são aceitas nas competições?

As competições são livres. Você pode usar Python, Java, JavaScript, C++, C#, entre outras.

7. Vai ter premiação para os alunos?

Sim, equipamentos de informática, licenças de software e bolsas de estudo.

8. Posso formar equipe com pessoas de outras escolas?

Sim, incentivamos a mistura de talentos nos Hackathons!

9. Qual a idade mínima para participar das competições?

14 anos (com autorização dos responsáveis para menores de 18).

10. Onde fica o edital de regras das competições?

O regulamento será enviado para o seu e-mail assim que sua inscrição for confirmada.
